# SPADExp
Simulator of photoemission angular distribution for experiment
(originally named as calcPSF, but the name is changed to SPADExp)

# Requirements
## C, C++
- make
- HDF5 (compiled with the ```--enable-cxx``` option)
- OpenMP

## Python3
- PyQt5
- pyqtgraph
- h5py
- numpy
- scipy
